## 如何阅读一本书

1. 阅读前：从 issues 中选择一本想读的（如无例外，豆瓣评分低于 8.0 的书，不读）
2. 阅读中：
   1. 读：求质不求量。感兴趣就慢慢读，觉得不值得读立即放弃，见 `3.1.1`
   2. 记：对感兴趣的内容标记
   3. 问：不懂的内容，搜一搜
   4. 拓：从本书内容，拓展到其他书籍、资料、纪录片等
3. 阅读后：
   1. 总结：写笔记（可选）
   2. 标记：
      1. 在 issue 中标记读完或者放弃
      2. 在对应的阅读大纲中标记读完
   3. 回到 `1.` 开始下一本
